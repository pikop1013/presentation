# Web Presentation Deck

ブラウザで使える「パワーポイント風」の発表資料テンプレートです。  
静的ファイルだけで動作し、矢印キーまたはボタンでスライドを切り替えできます。

## 使い方

1. `index.html` をブラウザで開く
2. `◀ / ▶` ボタン、またはキーボードの `← / →` で移動
3. `index.html` の各 `<section class="slide">` を編集して資料をカスタマイズ

## ファイル構成

- `index.html`: スライド本体
- `styles.css`: デザイン（ダークテーマ、レスポンシブ対応）
- `script.js`: スライド遷移ロジック

## ローカル確認

以下のどちらかで確認できます。

- ファイルを直接開く（ダブルクリック）
- 簡易サーバーを起動する

```bash
python3 -m http.server 8000
```

その後、`http://localhost:8000` を開いてください。
